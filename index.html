Инструкция для Google Drive:

  1. Загрузите видео на Google Drive
  2. Правой кнопкой мыши → "Предоставить доступ" → "Доступ по ссылке"
  3. Скопируйте ID файла из ссылки (после /d/ и до /view)
  4. Используйте ссылку: https://drive.google.com/file/d/ВАШ_ID/preview

Инструкция для Яндекс.Диск:

  1. Загрузите видео на Яндекс.Диск
  2. Правой кнопкой мыши → "Поделиться" → "Открыть доступ"
  3. Получите прямую ссылку через сервис yadi.sk/direct
  4. Используйте полученную прямую ссылку

Инструкция для Dropbox:

  1. Загрузите видео на Dropbox
  2. Правой кнопкой мыши → "Поделиться" → "Создать ссылку"
  3. Измените в ссылке "www.dropbox.com" на "dl.dropboxusercontent.com"
  4. Уберите параметры из ссылки, оставив только основную часть

Инструкция для Mail.ru Cloud:

  1. Загрузите видео в Облако Mail.ru
  2. Правой кнопкой мыши → "Поделиться" → "Скопировать публичную ссылку"
  3. Используйте полученную ссылку
